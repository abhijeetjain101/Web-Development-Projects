HTML Structure

This theme is a flexible layout with max 3 column. All of the information within the main content area is nested within a div with a class of "main class". You have to open index page in order to change it.

If you would like to edit the color, font, or style of any elements in this template, you could do by using "Inspect Element" on Chrome, Safari, Mozilla and Opera(press right click on browser ).  It is really powerful tool to work with html and css codes

 

CSS Files and Structure

I'm using 2 CSS files in this template. The first file (style.css) contains all of the specific stylings for the page. The file is separated into sections. The second file (style_2.css) contains contact page css.

 

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

 

JavaScripts

All call actions are run in main.js

Attention : If you remove any of javascripts, please don't forget to remove js code call action that belongs to removed js from main.js . Otherwise your site works inproperly

Other JavaScript files:

  • main.js - includes call actions
  • scrollreveal.min.js - used for smooth scrooll behaviour
  • jswiper-bundle.min.js - used for swipe to different options
  • mixitup.min.js - MixItUp 3 A high-performance, dependency-free library for animated filtering, sorting, insertion, removal and more...
